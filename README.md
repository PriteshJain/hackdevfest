## About Project

Trivia questions with many of topics free! Have a fun time online. Film, Music, History, and much more.

## Getting Started

```sh
$ npm install
// or
$ yarn
```

Run in dev mode:

```sh
$ npm run dev
// or
$ yarn dev
```
